- Explica qué es GitHub y qué relación tiene con Git.
- 
Github es una plataforma en la nube que contiene proyectos que utilizan Git, que guarda cada cambio que haces en el codigo de algun proyecto. Git gestiona tus proyectos y luego se suben a Github para compartirlos

-Describe los tipos de repositorios que existen en GitHub.
publicos: cualquier persona puede acceder a el
privados: solo tu y las personas colaboradoras pueden ver que hay dentro
fork: copias de repositorios ajenos que puedes editar en tu cuenta
